# Quebra de senha

Programa usado para quebrar uma senha de 8 caracteres numéricos.

## Comandos

1. Gerar arquivo executável:

   ```bash
   make

2. Sequencial:

   ```bash
   ./seq 99999999
   ```
3. Pthread:

   ```bash
   ./par_pthread 99999999 4
   ```

4. OpenMP:
   ```bash
   ./par_openmp 99999999 4
   ```
